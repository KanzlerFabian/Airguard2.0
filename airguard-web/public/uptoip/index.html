Demo-Hinweis

Warum Beispieldaten?

Die AWS-Demo nutzt realistische Beispieldaten, damit das Dashboard unabhängig von ESP32, Raspberry Pi, Prometheus und lokalen Netzwerkdiensten zuverlässig gezeigt werden kann. Die Demo ersetzt nicht die Live-Anbindung, sondern macht das Projekt präsentierbar.
