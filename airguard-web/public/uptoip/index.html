Warum Beispieldaten?

Die AWS-Demo nutzt realistische Beispieldaten, damit das Dashboard unabhängig von ESP32, Raspberry Pi und lokalen Diensten zuverlässig gezeigt werden kann. Die Live-Anbindung bleibt als Architekturidee erhalten.
